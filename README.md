MetadaneShapefile v1.2-beta, 9 kwietnia 2013
---
Opracowanie metadanych dla ortofotomapy w postaci pliku ESRI Shapefile

# Funkcje

* edytor schematu
* import zasiêgów zdjêæ z Microstation v7 (*.dgn)
* import zasiêgów zdjêæ z ESRI Shapefile (*.shp)
* import zasiêgów zdjêæ z Photo file (footprint) (*.txt)
* import punktów œrodków zdjêæ i atrybutów z plików Excel (*.xls)
* import atrybutów/metadanych dla zasiêgów z plików Excel (*.xls)
* import takich samych atrybutów dla wszystkich zdjêæ (kolumnowo) z plików Excel (*.xls)
* eksport metadanych do Shapefile (*.shp)
* edytor atrybutów/metadanych
